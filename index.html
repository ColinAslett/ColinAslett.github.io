A Brief Comparison of the Rating Systems Used

Elo: The Elo rating system is well known for its use in chess. Elo uses the existing rating of the competitors and the result of the match to determine each competitor's new Elo rating. The Elo rating for each team starts at 1500 at the beginning of the season with a global K-value of 80-100. A more detailed description of the Elo rating system can be found here: Elo Details

Glicko 2: The Glicko 2 rating system is similar to the Elo system. The difference is mostly the inclusion of a ratings deviation that gives an indication of the reliability of the rating. The rating deviation for each team starts at 350 but, as more games are played, Glicko 2 gets more confident and it will decrease. A more detailed description of this rating system can be found here: Glicko 2 Details

HITS: Hyperlink-Induced Topic Search (HITS) was originally designed as a link analysis algorithm that rates web pages. I have modified the algorithm to rate college football teams. A more detailed description of this rating system can be found here: HITS Details

PageRank: PageRank was originally developed by Google to rank websites but can be applied to sports as well. As shown in Table 1, it is a graph-based rating system and the rating of a team is affected by how the strength of their opponents continue to evolve over the course of the season after they have played them. A more detailed description of this rating system can be found here: PageRank Details

BG BG is my interpretation of the BeatGraph system. As shown in Table 1, this is also a graph based rating system. BeatGraph is the only rating system I use that purposefully destroys part of the game data in an attempt to create a acyclic graph. In college football this generally results in about 30-40% of game data being thrown away. A more detailed description of this rating system can be found here: BeatGraph Details

An Example using the 2020-2021 Season

Table 2 shows the results obtained at the conclusion of the 2020/2021 college footbal regular season based on all of the games played to that point. The results for each rating system are listed in their rank order and shown next to each other to make it easy to see the differences. The final ranking from the BCS committee is also shown for reference. The program generates a rating for all college Div. 1 football teams but only the top 25 for each system are shown in Table 2.

Notes: In 2020/2021, with Covid-19 forcing many conferences to only play against teams within the conference, the graph-based rating systems suffered. Alabama had a perfect score in the SEC but was ranked lower than other teams that simply played teams outside of their conference. Another challenge for the rating systems in the 2020/2021 season was that many teams did not play the same number of games, which is crucial when the regular season is only about a dozen games to begin with.

Applying MLR to Predict Game Results

Win Probabilities using MLR: In addition to providing the relative ranking of college football teams, I will also provide the win probabilities for upcoming games using the Multiple Linear Regression(MLR) formula. As shown in Table 3, MLR will use the scores of the past games and game data (like yards per game) to determine the win probabilities of a team in a future match. Table 4 shows an example of how I will record the percentage odds of a given team winning their match starting in the 2021/2022 season. The table will be updated weekly before the games begin once the college season begins. A more detailed description of this rating system can be found here: MLR Details
